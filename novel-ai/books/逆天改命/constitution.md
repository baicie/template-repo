# 创作宪法

> 本文件是所有创作的**最高原则**，不可违背。定义不可妥协的质量底线、风格规范和读者承诺。

---

## 核心价值观

### 故事至上

```
每个章节必须推进故事
- 无意义的日常要删除
- 水文要杜绝
- 节奏要紧凑
```

### 角色立体

```
人物要有自己的目标和动机
- 行为符合性格设定
- 对话符合身份背景
- 成长要有轨迹
```

### 世界观一致

```
设定一旦确立不能随意修改
- 力量体系要平衡
- 规则要贯彻始终
- 不能为剧情强行改设定
```

### 语言精炼

```
用最少的字传达最多信息
- 避免废话和重复
- 描写要有画面感
- 对话要推进剧情
```

### 情感真实

```
情感发展要有逻辑
- 不能强行煽情
- 转折要有铺垫
- 让读者产生共鸣
```

---

## 质量底线（绝不可妥协）

1. **逻辑自洽**：情节逻辑、角色行为逻辑、世界观逻辑必须自洽
2. **人设一致**：角色性格、对话风格、能力边界必须一致
3. **设定不矛盾**：世界观规则不能前后矛盾
4. **伏笔必回收**：埋下的伏笔必须回收
5. **爽点兑现**：承诺的爽点必须兑现

---

## 风格规范

### 文风要求

- **句式**：短句为主，善用句号，切忌长句堆砌
- **对话**：直接用冒号，不用"说道"等引语词
- **描写**：动作精准，心理克制，画面感强
- **节奏**：快节奏，强爽点，不拖沓

### 禁用词

- 突然、瞬间、只见、但见、不由得、情不自禁
- 嘴角勾起、眸光一闪、淡淡道、缓缓道

### 字数规范

| 章节类型 | 推荐字数 |
|---------|---------|
| 日常铺垫章 | 2000-3000 字 |
| 标准章节 | 3000-5000 字 |
| 高潮/战斗章 | 4000-6000 字 |
| 过渡章 | 1500-2500 字 |

---

## 章节结构

1. **开头**：承上启下，不废话，直接进入
2. **发展**：有节奏变化，小高潮不断
3. **高潮**：情绪爆发，爽点密集
4. **结尾**：留悬念或情绪高点，禁止"作者有话说"

---

## 读者承诺

- 每章至少 1 个小高潮或转折
- 每章结尾必有钩子
- 主角不能无故消失超过 3 章
- 配角的死亡/退场必须有意义
- 情节推进优先于描写堆砌
