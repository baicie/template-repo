# 个人写作规范

## 文风偏好

- 战斗描写：简洁有力，动作精准，过程不要太长
- 情感描写：细腻但不过度，内心独白控制在 100 字内
- 对话：不写"说道"等引语词，直接用冒号
- 旁白：尽量通过行动和对话展示，少用大段说明

## 章节结构

- 每章至少 2 个小高潮或转折点
- 章末留悬念或情绪高点
- 日常章节不超过 3 章连续
- 战斗章节后至少 1 章缓冲

## 字数参考

| 章节类型 | 推荐字数 |
|---------|---------|
| 日常铺垫章 | 2000-3000 字 |
| 标准章节 | 3000-5000 字 |
| 高潮/战斗章 | 4000-6000 字 |
| 过渡章 | 1500-2500 字 |

## 禁用词

| 禁用词 | 原因 | 替代建议 |
|--------|------|---------|
| 突然 | 太突兀 | 删除或改用具体时间 |
| 瞬间 | 同上 | 改用具体动作 |
| 紧接着 | 节奏太赶 | 改用自然过渡 |
| 不由得 | 心理描写滥用 | 用表情/动作代替 |
| 情不自禁 | 同上 | 同上 |
| 只见 | 网文老套路 | 直接描写场景 |
| 但见 | 同上 | 同上 |
| 嘴角勾起 | 用烂的描写 | 具体神情 |
| 眸光一闪 | 同上 | 具体眼神变化 |

## 章节类型定义

| 类型 | 说明 |
|------|------|
| 日常 | 节奏放缓，铺垫情感，信息交代 |
| 战斗 | 动作精准，简洁有力，高潮处放大 |
| 情感 | 内心细腻，对话自然，情感层次递进 |
| 转折 | 意外展开，节奏突变，吸引继续读 |
| 高潮 | 情绪爆发，爽点密集，信息量加大 |

## 钩子类型

| 类型 | 写法 |
|------|------|
| 悬念钩 | 章末提出疑问，吸引继续读 |
| 爽点钩 | 高潮处戛然而止，让人欲罢不能 |
| 危机钩 | 主角陷入危险，读者担心 |
| 情感钩 | 情感升温时戛然而止 |
| 反转钩 | 出人意料的结尾 |
