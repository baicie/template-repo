# 从灵感到成书 — 完整工作流

## 概览

```
构思期（1-3天）
    ↓
世界观构建（3-7天）
    ↓
角色塑造（2-5天）
    ↓
大纲规划（3-7天）
    ↓
正文写作（持续）
    ↓
修订润色（完成初稿后）
```

---

## 阶段一：构思期

### 1. 确定核心创意

回答以下问题：
- 你想写什么类型的小说？（玄幻/都市/仙侠/科幻...）
- 核心卖点是什么？（废柴逆袭/系统流/穿越/...）
- 读者为什么会喜欢这本书？

### 2. 创建新书

1. 在 `books/` 下创建新书目录
2. 填写 `meta.md`
3. 填写 `00-context.md`
4. 激活书籍：`configs/active-book.md`

### 3. 工具推荐

使用 `prompts/inspiration/01-scenario-gen.md` 生成创意方向。

---

## 阶段二：世界观构建

### 顺序

```
1. genre-analysis.md   → 确定类型惯例
2. world-rules.md     → 世界运行规则
3. power-system.md    → 力量体系（最核心）
4. geography.md       → 地理设定
5. factions.md        → 势力派系
6. glossary.md        → 术语词典
```

### 输出位置

所有世界观内容存入 `books/{书名}/00-world.md`

### 核心：力量体系设计

这是网文最关键的设定，决定了战斗爽感和升级节奏。

重点考虑：
- 等级数量（7-9 级为宜）
- 每个等级的标志性特征
- 升级条件和代价
- 越级战斗的可能性

---

## 阶段三：角色塑造

### 顺序

```
1. protagonist.md   → 设计主角
2. antagonist.md   → 设计核心反派
3. supporting.md   → 设计重要配角
4. character-web.md → 构建关系网
5. speaking-style.md → 提炼话风
```

### 输出位置

所有角色内容存入 `books/{书名}/00-characters.md`

### 重点

- 主角必须有记忆点和成长弧线
- 反派必须有合理的动机
- 配角要有独特功能

---

## 阶段四：大纲规划

### 顺序

```
1. story-arc.md       → 主线大纲
2. arc-breakdown.md   → 章节分配
3. plot-twist.md      → 高潮设计
4. subplot.md         → 支线铺排
5. chapter-outline.md → 单章大纲
```

### 输出位置

大纲存入 `books/{书名}/00-outline.md`

### 大纲粒度

- 主线：到卷级别
- 支线：到关键节点
- 单章：在开始写之前才写

---

## 阶段五：正文写作

### 流程

```
1. 更新 00-context.md（本章信息）
2. 读取相关设定
3. 开始写作
4. 完成草稿
5. 润色（可选）
6. 一致性检查（可选）
7. 存入 chapters/
8. 更新 00-context.md（角色状态、情节推进）
```

### 写作节奏

| 类型 | 字数 | 频率 |
|------|------|------|
| 日更 | 3000-5000 字 | 每天 |
| 隔日更 | 5000-8000 字 | 每两天 |
| 周更 | 10000-15000 字 | 每周 |

---

## 阶段六：修订润色

### 流程

```
1. consistency-check.md   → 全书一致性检查
2. pacing-review.md      → 节奏审视
3. character-consistency.md → 角色一致性
4. grammar-check.md      → 语病检查
```

### 修订时机

- **小修**：每章写完后立即检查
- **大修**：写完一个卷后通读
- **终修**：全书完成后整体审视
