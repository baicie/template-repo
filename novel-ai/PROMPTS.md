## 模板变量速查

| 变量 | 说明 |
|------|------|
| `{{book_title}}` | 书名 |
| `{{genre}}` | 小说类型 |
| `{{current_chapter}}` | 当前章节号 |
| `{{word_count_target}}` | 目标字数 |
| `{{outline_node}}` | 本章大纲节点 |
| `{{world_rules}}` | 世界规则 |
| `{{characters}}` | 角色设定 |
| `{{previous_content}}` | 前文结尾 500 字 |
| `{{scene_type}}` | 场景类型 |
| `{{pacing}}` | 节奏要求 |
