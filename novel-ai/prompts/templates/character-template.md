# 人设速成模板

> 网文人设的核心不是档案，而是**剧情发动机**。每个角色必须能制造冲突、提供信息、推动剧情。

---

## 一、好人设的六个维度

```
欲望：他想要什么？
恐惧：他怕失去什么？
能力：他靠什么解决问题？
缺陷：他为什么会犯错？
关系：他和主角有什么利益冲突？
剧情作用：他推动哪条线？
```

## 六、人物关系速写

写人物关系时，问三个问题：

```
1. 他和主角有利益冲突吗？
   └→ 有 → 是反派或对立势力
   └→ 无 → 是盟友或中立

2. 他有什么主角没有的东西？
   └→ 情报/资源/能力/背景 → 他有利用价值

3. 他会因为什么背叛/离开主角？
   └→ 利益不够 / 威胁太大 / 信念不同 → 这就是他的缺陷
```

## 七、话风提炼

每个角色必须有**一句标志性口头禅**和**一种说话节奏**：

```
| 角色 | 口头禅 | 节奏 | 示例 |
|------|--------|------|------|
| 主角 | "既然..." | 慢→快 | "既然你不讲道理，那就别怪我动手了。" |
| 反派 | "你以为..." | 傲慢 | "你以为突破了一个小境界，就能和我抗衡？" |
| 师兄 | "师弟..." | 温和 | "师弟，此事还需从长计议。" |
| 师妹 | "哼！" | 直接 | "哼，我才不怕他呢！" |
```

## 八、快速人设生成

遇到卡壳时，用这三个问题快速生成一个配角：

```
问题1：主角现在缺什么？
└→ 缺情报 → 安排一个情报贩子
└→ 缺资源 → 安排一个商人
└→ 缺力量 → 安排一个师父或前辈

问题2：主角现在最怕什么？
└→ 怕亲人受伤 → 安排一个要保护的家人
└→ 怕暴露身份 → 安排一个可能识破的人
└→ 怕打不过 → 安排一个更强的导师

问题3：谁能让剧情更好看？
└→ 一个两面三刀的朋友
└→ 一个亦敌亦友的势力
└→ 一个表面冷漠内心关心的角色
```
