# 输出格式模板

```markdown
# 输出格式模板

## 章节正文格式

```markdown
# 第 {{chapter_number}} 章 {{chapter_title}}

正文内容...

---
```

### 说明
- 章节标题使用 Markdown 一级标题
- 章节之间用 `---` 分隔
- 正文直接输出，不加额外格式

## 分析/建议类输出格式

### 问题分析
```markdown
# 问题分析

## 问题 1
- **描述**：
- **位置**：
- **建议**：

## 问题 2
（同上）
```

### 建议输出
```markdown
# 建议

## 方向 1：{{direction}}
- **核心内容**：
- **优势**：
- **风险**：

## 方向 2：{{direction}}
（同上）
```

## 设定类输出格式

```markdown
# {{设定名称}}

## 基本信息
- **名称**：
- **类型**：
- **描述**：

## 详细设定
{{content}}

## 相关内容
- 相关角色：
- 相关势力：
- 相关地点：
```

## 检查报告格式

```markdown
# {{检查类型}}报告

## 基本信息
- 章节：第 {{chapter}} 章
- 字数：约 {{word_count}} 字
- 检查时间：{{date}}

## 检查结果
{{content}}

## 问题汇总
| 问题 | 位置 | 严重程度 |
|------|------|---------|
| {{issue}} | {{location}} | {{severity}} |

## 总体评价
【{{rating}}】
```

## 输出规范
- 正文直接输出纯文本
- 用 Markdown 标题标记章节
- 非正文输出使用 Markdown 格式
- 需要决策的内容用【作者决定】标注
- 发现问题时，明确指出位置和修改建议
```
