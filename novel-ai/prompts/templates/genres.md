# 题材模板库

> 内置常见网文题材的快速启动模板。每个模板包含：核心卖点、开篇结构、爽点节奏、经典金手指。

---

## 1. 玄幻废柴流

### 核心公式

```
废柴 → 金手指 → 修炼 → 打脸 → 升级 → 更大舞台
```

### 卖点

```
- 废柴逆袭的代入感
- 升级体系的可见进步
- 打脸的爽快感
- 地图扩张的史诗感
```

### 金手指类型

```
| 类型 | 说明 |
|------|------|
| 系统流 | 任务系统、签到系统 |
| 重生流 | 前世记忆、预知未来 |
| 传承流 | 老爷爷、神器、秘境 |
| 吞噬流 | 吸收他人力量 |
```

### 开篇结构

```
第一章：废物身份 + 当众羞辱 + 觉醒金手指
第二章：第一次修炼 + 小试身手
第三章：打脸第一个敌人 + 更大敌人出现
```

### 爽点节奏

```
| 间隔 | 爽点类型 |
|------|---------|
| 每3-5章 | 打脸小反派 |
| 每10章 | 获得重要资源 |
| 每30章 | 击败阶段性Boss |
| 每100章 | 地图切换 |
```

## 2. 都市重生流

### 核心公式

```
重生 → 知道未来 → 利用信息差 → 改变命运 → 登顶
```

### 卖点

```
- 信息差的爽感（知道未来）
- 弥补遗憾的情感共鸣
- 复仇的快感
- 商战/权谋的智斗
```

### 金手指类型

```
| 类型 | 说明 |
|------|------|
| 重生200X年 | 利用时代红利 |
| 重生到家人还活着 | 救亲人 |
| 重生到危机前 | 提前布局 |
| 重生到末世前 | 囤物资、建基地 |
```

### 开篇结构

```
第一章：重生回关键时间点 + 前世遗憾
第二章：制定改变计划 + 立即行动
第三章：第一次小成功 + 遇到前世仇人
```

### 爽点节奏

```
| 类型 | 爽点 |
|------|------|
| 信息差爽点 | 用未来知识碾压对手 |
| 救亲爽点 | 救回前世失去的人 |
| 打脸爽点 | 仇人还不知道他能未卜先知 |
| 布局爽点 | 前世仇人一步步落入陷阱 |
```

## 3. 诡异修仙流

### 核心公式

```
修仙 + 诡异元素 → 升级有代价 → 探索有危险 → 步步为营
```

### 卖点

```
- 诡异悬疑的紧张感
- 升级代价的戏剧性
- 世界观的神秘感
- 探索发现的惊喜
```

### 世界观规则示例

```
世界核心规则：所有修行都需要供奉神明，但供奉越深越容易被污染

升级方式：偷学禁忌之法
升级代价：每次突破丢失一段真实记忆
升级风险：被神明发现，被当成异端追杀
```

### 开篇结构

```
第一章：诡异征兆 + 家人出事 + 接触禁忌
第二章：获得诡异能力 + 付出代价
第三章：能力初显 + 更大的危险浮出水面
```

### 爽点节奏

```
| 类型 | 爽点 |
|------|------|
| 能力爽点 | 解锁新诡异能力 |
| 信息爽点 | 发现世界真相 |
| 智斗爽点 | 用诡异能力算计敌人 |
| 反转爽点 | 原本的危险变成机遇 |
```

## 4. 都市系统流

### 核心公式

```
普通人 + 系统任务 → 做任务 → 获得奖励 → 变强
```

### 卖点

```
- 任务完成的成就感
- 系统奖励的新鲜感
- 日常变强的爽感
- 任务失败的紧张感
```

### 系统类型

```
| 系统 | 特点 |
|------|------|
| 神豪系统 | 花钱返利、狂薅羊毛 |
| 医术系统 | 治病救人、声名鹊起 |
| 直播系统 | 粉丝值换技能 |
| 捡钱系统 | 捡东西获得奖励 |
| 宠兽系统 | 收服培养召唤兽 |
```

### 开篇结构

```
第一章：普通人困境 + 系统激活
第二章：第一个任务 + 获得奖励
第三章：能力展示 + 遇到贵人/敌人
```

## 5. 仙侠修真流

### 核心公式

```
凡人 → 踏入修真 → 宗门纷争 → 成仙之路
```

### 卖点

```
- 仙气飘飘的想象空间
- 功法神通的千变万化
- 宗门派系的权力斗争
- 长生不老的终极追求
```

### 力量体系

```
境界：炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 大乘 → 渡劫

势力：宗门 → 世家 → 王朝 → 散修联盟
资源：灵石 + 天材地宝 + 功法秘籍
```

### 开篇结构

```
第一章：山村少年 + 偶遇仙人 + 被带离家乡
第二章：宗门入门 + 遭受打压 + 意外发现秘密
第三章：小试身手 + 获得认可 + 更大的阴谋浮现
```

## 6. 悬疑推理流

### 核心公式

```
案件 → 线索 → 推理 → 真相 → 反转
```

### 卖点

```
- 推理的智力快感
- 真相揭露的震撼
- 凶手的意外性
- 背后更大阴谋的悬念
```

### 常见结构

```
第一章：案件发生 + 侦探登场
第二章：收集线索 + 排除嫌疑
第三章：推理成型 + 第一次反转
```

## 7. 融合题材设计

### 常见融合公式

```
| 主类型 | 融合 | 卖点 |
|--------|------|------|
| 玄幻 | + 穿越 | 现代知识降维打击 |
| 都市 | + 重生 | 弥补遗憾 + 信息差 |
| 玄幻 | + 系统 | 任务驱动 + 可见成长 |
| 仙侠 | + 诡异 | 升级有代价 + 神秘感 |
| 都市 | + 异能 | 普通人变超能力者 |
| 玄幻 | + 星际 | 升级 + 宇宙探索 |
```

### 融合注意事项

```
1. 先确定主菜，再加调料
2. 融合元素不能喧宾夺主
3. 核心爽点必须来自主菜
4. 融合元素要服务于核心爽点
```

## 题材选择决策树

```
想写爽文还是慢热？
├── 爽文 → 选都市/玄幻
└── 慢热 → 选悬疑/诡异

主角想要什么？
├── 变强复仇 → 玄幻废柴流
├── 弥补遗憾 → 都市重生流
├── 探索真相 → 悬疑推理流
└── 长生不老 → 仙侠修真流

想写长篇还是短篇？
├── 长篇（100万字+）→ 玄幻/都市/仙侠
└── 短篇（10-30万字）→ 系统流/重生流
```
