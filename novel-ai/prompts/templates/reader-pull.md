# 追读力系统

> 来自 webnovel-writer 的追读力设计。核心问题：读者为什么愿意追更？通过 Hook（钩子）、Cool-point（冷却点）和 Debt（债务）三个机制管理读者期待。

## 核心公式

```
追读力 = 钩子强度 - 冷却程度 + 债务压力
```

- **钩子**：吸引读者继续读
- **冷却**：读者想要暂停的情绪
- **债务**：承诺读者的伏笔/冲突，必须兑现

## Hook（钩子）

### 钩子类型

| 类型 | 描述 | 强度 | 示例 |
|------|------|------|------|
| 悬念钩 | 提出问题，读者想知道答案 | 强 | "他不知道的是..." |
| 冲突钩 | 矛盾即将爆发 | 强 | "三天后的比武场上..." |
| 爽点预告 | 预告即将到来的爽点 | 中 | "更大的机缘在等着他" |
| 情感钩 | 情感高潮即将到来 | 中 | "她欲言又止..." |
| 信息钩 | 揭示新信息 | 弱 | "原来这一切都是..." |

### 每章钩子规则

- 每章必须有至少 1 个强钩子
- 章节结尾必须留钩子
- 章节开头可以承接上章钩子

### 钩子追踪

```json
{
  "hooks": [
    {
      "id": "hook-001",
      "type": "悬念钩",
      "content": "萧寒的玉佩有什么秘密？",
      "planted": { "chapter": 1, "location": "结尾" },
      "status": "active",
      "revealPlanned": { "chapter": 10, "description": "玉佩是上古传承" }
    }
  ]
}
```

## Cool-point（冷却点）

### 冷却点类型

| 类型 | 描述 | 影响 |
|------|------|------|
| 重复战斗 | 类似的战斗场景反复出现 | 中 |
| 日常铺垫 | 过长的日常场景 | 中 |
| 支线拖延 | 支线占用过多篇幅 | 中 |
| 信息解释 | 过多的设定解释 | 高 |
| 节奏拖沓 | 章节推进缓慢 | 高 |

### 冷却点监控

每章完成后评估：
- 本章是否有冷却点？
- 冷却点持续多久？
- 是否影响追读意愿？

### 冷却点公式

```
冷却程度 = Σ(冷却点强度 × 持续比例)
```

- 冷却程度 > 50%：危险，需要立即调整
- 冷却程度 > 30%：警告，需要注意
- 冷却程度 < 30%：正常

## Debt（债务/伏笔）

### 债务类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 承诺型 | 角色做出的承诺 | "我会回来找你" |
| 冲突型 | 已建立但未解决的矛盾 | 退婚约定 |
| 悬念型 | 提出的疑问 | 玉佩的来历 |
| 期待型 | 铺垫的爽点 | 系统即将升级 |

### 债务管理

#### 债务评级

| 评级 | 说明 | 兑现期限 |
|------|------|---------|
| 微型债务 | 1-2 章内 | 第 X 章 |
| 小型债务 | 3-5 章内 | 第 X 章 |
| 中型债务 | 5-10 章内 | 第 X 章 |
| 大型债务 | 10+ 章 | 第 X 卷 |

#### 债务追踪

```json
{
  "debts": [
    {
      "id": "debt-001",
      "type": "冲突型",
      "content": "三月之约",
      "planted": { "chapter": 1 },
      "due": { "chapter": 30 },
      "status": "active",
      "severity": "large",
      "promisedPayoff": "萧寒当众击败林若雪"
    },
    {
      "id": "debt-002",
      "type": "期待型",
      "content": "系统升级预告",
      "planted": { "chapter": 1 },
      "status": "active",
      "severity": "small"
    }
  ]
}
```

### 债务红线

- **微型债务**：最多同时存在 5 个
- **小型债务**：最多同时存在 3 个
- **大型债务**：最多同时存在 2 个
- **超过红线**：必须先兑现，再新增债务

### 债务兑现检查

```
【债务状态报告】

进行中：
- debt-001 三月之约：第 1 章 → 应在第 30 章兑现，剩余 X 章
- debt-002 系统升级：第 1 章 → 应在第 5 章兑现，剩余 X 章

已到期未兑现：
- debt-XXX（警告！需要立即兑现）

已兑现：
- debt-YYY：已兑现（第 10 章）
```

## 追读力评分

```
追读力评分 = 钩子总分 - 冷却总分 + 债务压力分

- 钩子总分：Σ(钩子强度 × 已铺垫程度)
- 冷却总分：Σ(冷却点强度 × 持续比例)
- 债务压力分：Σ(债务额度 × 逾期比例)

评分标准：
- 80-100：极强追读力
- 60-79：强追读力
- 40-59：中等追读力
- 20-39：弱追读力
- <20：危险，读者可能弃书
```
