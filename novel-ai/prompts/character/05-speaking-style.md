# 话风提炼

```markdown
# 话风提炼

## 任务
为「{{book_title}}」中的重要角色提炼专属话风。

## 已有角色
{{character_list}}

## 提炼角色：{{character_name}}

## 基础信息
- **身份**：{{identity}}
- **性格**：{{personality}}
- **背景**：{{background}}

## 话风要素

### 语言特点
- **【语速】** 快/慢/适中/根据情绪变化
- **【音量】** 高亢/低沉/根据场合变化
- **【用词】** 书面/口语/古风/网络/专业术语
- **【句式】** 短句为主/长句/混合/喜欢用排比
- **【语气词】** 常用的语气词（如：啊、呢、吧、嘛）
- **【口头禅】** 反复出现的表达

### 说话风格
- **【直接/委婉】** 直接表达还是绕弯子
- **【幽默/严肃】** 经常开玩笑还是一本正经
- **【攻击性】** 说话带刺还是温和
- **【自信度】** 自信满满/谦虚低调/自卑掩饰

### 特色表达
- **【专属词汇】** 这个角色特有的说法
- **【习惯用语】** 在特定场景下的固定表达
- **【禁忌话题】** 这个角色不愿意提及的话题

## 对话示例

### 场景 1：日常对话
```
{{other_char}}：「{{question}}」
{{char_name}}：「{{response}}」
```

### 场景 2：情绪激动时
```
{{other_char}}：「{{provocation}}」
{{char_name}}：「{{angry_response}}」
```

### 场景 3：关键对话
```
{{other_char}}：「{{key_question}}」
{{char_name}}：「{{key_response}}」
```

## 话风描写片段
描写这个角色说话时的整体感觉：
{{description}}

## 提炼要点
1. **话风要符合性格**：说话方式是性格的外在表现
2. **话风要符合身份**：说什么话还要看是什么身份
3. **话风要有辨识度**：读者一听就知道是谁说的话
4. **话风可以变化**：在不同情绪下说话方式可以变化
5. **避免脸谱化**：不要让每个角色说话都一样
```
