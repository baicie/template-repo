# 七步方法论索引

## 概述

本目录按七步方法论的顺序组织提示词模板。

### 七步方法论

```
1. constitution  → 创作宪法
2. specify      → 故事规格
3. clarify      → 澄清决策
4. plan         → 创作计划
5. tasks        → 任务分解
6. write        → 章节写作
7. analyze      → 综合验证
```

## 文件索引

| 步骤 | 文件 | 说明 |
|------|------|------|
| 1. constitution | `01-constitution.md` | 建立不可违背的创作原则 |
| 2. specify | `02-specify.md` | 像 PRD 一样定义故事 |
| 3. clarify | `03-clarify.md` | 通过问答澄清模糊点 |
| 4. plan | `04-plan.md` | 制定技术实现方案 |
| 5. tasks | `05-tasks.md` | 分解为可执行任务 |
| 6. write | `../writing/01-chapter-draft.md` | 基于任务写作 |
| 7. analyze | `07-analyze.md` | 七维度质量验证 |
