```markdown
# 任务分解

## 任务
将创作计划分解为可执行的任务清单。

## 已有信息
- 书名：{{book_title}}
- 创作计划：{{creative_plan_summary}}

## 任务框架

### 创作准备阶段 [P0]
| 任务 | 优先级 | 依赖 | 说明 |
|------|-------|------|------|
| 完成世界观设定 | [P0] | 无 | 力量体系、地理、势力 |
| 完成主角设定 | [P0] | 无 | 包括金手指、成长弧线 |
| 完成大纲规划 | [P0] | 世界观、主角 | 到章节级别 |
| 建立追踪系统 | [P0] | 无 | JSON 文件初始化 |

### 世界观构建任务 [P1]
| 任务 | 优先级 | 依赖 | 说明 |
|------|-------|------|------|
| 设计力量体系 | [P1] | 无 | 境界设定、升级条件 |
| 设计地理设定 | [P1] | 无 | 主要地点、势力分布 |
| 设计势力派系 | [P1] | 力量体系 | 各势力关系 |
| 编写术语词典 | [P2] | 力量体系 | 术语整理 |

### 角色构建任务 [P1]
| 任务 | 优先级 | 依赖 | 说明 |
|------|-------|------|------|
| 设定主角 | [P1] | 无 | 详细档案 |
| 设定女主 | [P1] | 主角 | 至少 1 名 |
| 设定反派 | [P1] | 力量体系 | 核心反派 |
| 设定配角 | [P2] | 主要角色 | 3-5 名重要配角 |
| 构建关系网 | [P1] | 所有角色 | 关系图谱 |

### 大纲规划任务 [P1]
| 任务 | 优先级 | 依赖 | 说明 |
|------|-------|------|------|
| 设计主线大纲 | [P1] | 世界观、主角 | 到卷级别 |
| 设计支线规划 | [P2] | 主线 | 主要支线 |
| 分配章节 | [P1] | 主线 | 到章节级别 |
| 设计高潮 | [P1] | 主线 | 全书高潮点 |
| 铺设伏笔 | [P1] | 主线 | 伏笔分布 |

### 章节写作任务 [P0]
| 任务 | 优先级 | 依赖 | 说明 |
|------|-------|------|------|
| 写第 1 卷 | [P0] | 大纲完成 | 前 30 章 |
| 写第 2 卷 | [P0] | 第 1 卷 | 31-80 章 |
| 写第 N 卷 | [P0] | 前卷完成 | 后续卷 |

### 修订任务 [P1]
| 任务 | 优先级 | 依赖 | 说明 |
|------|-------|------|------|
| 伏笔检查 | [P1] | 第 1 卷完成 | 确认伏笔已铺 |
| 一致性检查 | [P1] | 每卷完成 | 设定一致性 |
| 节奏审视 | [P2] | 第 1 卷完成 | 节奏调整 |
| 整体润色 | [P2] | 初稿完成 | 文笔提升 |

## 任务标记说明

| 标记 | 含义 |
|------|------|
| [P0] | 必须首先完成 |
| [P1] | 正常推进 |
| [P2] | 可选完善 |
| [依赖:X] | 需要先完成 X |
| [P] | 可以同时进行 |

## 输出
输出任务清单，存入 `books/{书名}/tasks.md`。
```
