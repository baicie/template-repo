# 节奏审视

```markdown
# 节奏审视

## 任务
审视以下内容的节奏是否合适。

## 已有信息
- 书名：{{book_title}}
- 审视范围：第 {{start_ch}} 章 - 第 {{end_ch}} 章
- 总字数：约 {{word_count}} 字
- 章节类型：{{chapter_type}}

## 待审视内容
{{content}}

## 审视维度

### 1. 章节内节奏

#### 段落节奏
- 是否有连续长段落？（容易读起来累）
- 是否有连续短段落？（容易读起来乱）
- 长短段落是否交替？

#### 情绪节奏
- 情绪是否不断攀升？
- 是否有起伏变化？
- 高潮是否足够突出？

#### 信息节奏
- 信息密度是否合适？
- 是否有信息过载的区域？
- 是否有过渡性的平淡区域？

### 2. 章节间节奏

#### 章节类型分布
- 日常章是否连续超过 3 章？
- 战斗章后是否有缓冲？
- 情感章是否过于密集？

#### 高潮分布
- 高潮是否足够密集？
- 高潮之间的间隔是否合理？
- 是否有连续平淡期？

### 3. 整体节奏

#### 爽点节奏
- 爽点是否足够密集？
- 爽点类型是否多样化？
- 是否有预期之外的惊喜？

#### 悬念节奏
- 每章是否都有悬念？
- 悬念是否及时揭晓？
- 是否有效引导读者继续读？

## 节奏检查清单

```
【章节内】
[] 长短段落交替，不单调
[] 情绪有起伏，不平缓
[] 信息密度合适，不堆砌不空洞

【章节间】
[] 章节类型交替，不重复
[] 高潮分布合理，不稀疏不过密
[] 章末钩子有力，不平淡

【整体】
[] 爽点足够，有期待感
[] 节奏张弛有度，有呼吸感
[] 读完意犹未尽，有继续读的动力
```

## 节奏问题诊断

### 节奏过快
- 表现：事件推进太快，来不及消化
- 原因：段落太短，信息太多
- 建议：增加铺垫，拉长节奏

### 节奏过慢
- 表现：读起来无聊，没有推进
- 原因：过渡太多，爽点太少
- 建议：删减过渡，增加冲突

### 节奏混乱
- 表现：读起来混乱，不知所云
- 原因：场景切换太频繁，缺少过渡
- 建议：明确场景边界，增加过渡
