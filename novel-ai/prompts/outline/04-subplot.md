### 支线分布
- 支线不应过于频繁，每 10-15 章安排一个主要支线
- 支线要有明确的开始和结束
- 支线结局要有所交代

|------|---------|---------|------------|
| | | | |
| | | | |

### 伏笔管理
- 重要伏笔：需要明确埋设和回收
- 次要伏笔：可以自然埋设
- 假伏笔：用来误导读者（慎用）

## 支线设计要点
1. 支线要服务于主线，不能喧宾夺主
2. 支线要有自己的完整性
3. 支线要有独特价值（情感/信息/爽点）
4. 支线结局要有所交代，不能烂尾
5. 不要有太多支线同时进行
