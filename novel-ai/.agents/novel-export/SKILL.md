# 平台格式导出

## 触发场景

用户说"导出小说"、"转换为平台格式"、"番茄格式"、"起点格式"、"投稿格式"。

## 支持的平台

| 平台 | 格式要求 |
|------|---------|
| 番茄小说 | 纯文本，无空行，无缩进 |
| 起点中文网 | 中文全角空格缩进（2字符） |
| 飞卢小说网 | 中文全角空格缩进 |
| 晋江文学城 | 段落间空一行 |
| 七猫小说 | 纯文本，无特殊格式 |
| 知乎 | 段落间空一行，口语化 |

## 各平台格式规范

### 番茄小说

- 段落之间**不留空行**
- **不添加缩进**
- **删除所有 Markdown 标记**（##、**等）
- 章节标题格式：`第X章 标题`
- 特殊符号：→、←、↑、↓ 等可用

### 起点中文网

- 段落开头添加**全角空格**缩进（`　　`）
- 章节标题单独一行
- 段落之间**不留空行**
- 删除 Markdown 标记

### 晋江文学城

- 段落之间**空一行**
- 章节标题单独一行
- 删除 Markdown 标记
- 避免过多感叹号

### 知乎

- 段落之间**空一行**
- 口语化表达
- 可加入 emoji 增强阅读体验
- 标题简洁有力

## 导出流程

1. **扫描**：读取 `chapters/` 目录下的所有 `.md` 文件
2. **排序**：按文件名排序（v01-ch01 → v01-ch02 → v02-ch03）
3. **转换**：按目标平台格式逐文件转换
4. **输出**：生成导出文件或目录

## 注意事项

- **发布前必读**：导出后人工检查一遍格式
- **缩进字符**：起点用全角空格，不是半角空格
- **图片处理**：如果有图片链接，确认平台是否支持
- **敏感词**：导出后用平台审核工具检查敏感词
