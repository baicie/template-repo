# 开篇检查

## 触发场景

用户说"开篇检查"、"黄金三章"、"检查开篇"、"检查前三章"、"前300字"。

## 黄金三章标准

适用于检查前1-3章，特别关注第一章。

### 第一章：抓读者

| 检查项 | 标准 | 权重 |
|--------|------|------|
| 前300字有爆点 | 爆点必须在第1-3段出现 | 高 |
| 主角已登场 | 第1段或第2段出现主角 | 高 |
| 避免寡淡开场 | 不能以"今天"、"有一天"开头 | 中 |
| 避免信息轰炸 | 前300字不超过3个新概念 | 中 |
| 有冲突暗示 | 已经有或即将出现冲突 | 高 |
| 代入感建立 | 读者能快速代入主角视角 | 中 |

### 第二章：展设定

| 检查项 | 标准 | 权重 |
|--------|------|------|
| 有紧迫目标 | 主角有明确目标 | 高 |
| 有具体阻碍 | 阻碍明确且有压力 | 高 |
| 能力展示 | 主角能力有初步展示 | 中 |
| 世界观展开 | 力量体系/规则有暗示 | 中 |

### 第三章：出爽点

| 检查项 | 标准 | 权重 |
|--------|------|------|
| 有小爽点 | 至少一个小爽点 | 高 |
| 结尾有钩子 | 留下悬念 | 高 |
| 成长暗示 | 主角有成长变化 | 中 |
| 期待建立 | 读者想知道后续 | 高 |

## 评分标准

- 80-100分：优秀开篇，预计留存率高
- 60-79分：良好，个别项需优化
- 40-59分：及格，有明显问题
- 40分以下：较差，需要大幅修改
