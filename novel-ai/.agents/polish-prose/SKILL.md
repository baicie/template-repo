# Polish Prose — 文笔润色

## 润色类型

### 1. 对话润色

目标：让角色对话更贴合人物性格，自然生活化。

```
输入：角色原对话 + 角色设定
输出：润色后的对话
```

润色要点：
- 检查是否符合角色话风（毒舌/冷幽默/言简意赅/啰嗦等）
- 去除"说道""笑着说"等老套引语词
- 对话直接用冒号：「xxx」
- 符合角色身份和当前情境

### 2. 动作描写润色

目标：让动作描写精准有力，不拖沓。

润色要点：
- 删除冗余的修饰词
- 动作动词具体化（「打」→「一拳砸向」「掌刀斜劈」）
- 战斗动作简洁，3-5 个动作交代完一轮交锋
- 避免「突然」「瞬间」等突兀词

### 3. 心理描写润色

目标：内心戏细腻但不过度，情感真实。

润色要点：
- 内心独白控制在 100 字以内
- 用行动和表情代替心理描写
- 避免「不由得」「情不自禁」
- 情感层次递进，不要一步到位

### 4. 整体文笔打磨

对整段/整章进行综合润色，同时优化：
- 段落节奏
- 场景转换衔接
- 详略得当
- 消除语病和错别字

## 使用模板

### 对话润色模板

```markdown
# 对话润色

## 角色设定（从 00-characters.md 提取）
{{character_definition}}

## 原始对话
{{original_dialogue}}

## 场景
{{scene_context}}

## 润色要求
- 话风：{{speaking_style}}
- 符合角色当前情绪状态
- 自然，不生硬

## 输出
直接输出润色后的对话，标注角色名。
```

### 文笔打磨模板

```markdown
# 文笔润色

## 文风要求（从 meta.md 提取）
- 整体风格：{{overall_tone}}
- 段落节奏：{{pacing}}
- 重点描写：{{emphasis}}

## 个人禁用词（从 configs/writing-rules.md 提取）
{{banned_words}}

## 待润色内容
{{content}}

## 润色维度
1. 语句通顺，无语病
2. 节奏合适，不拖沓
3. 用词精准，不过度修饰
4. 风格统一，符合设定

## 输出
直接输出润色后的内容，同时在【】中简要说明修改理由。
```

## 润色原则

1. **不改原意**：润色不能改变情节和角色意图
2. **风格统一**：润色后的文字必须符合作品的文风定位
3. **作者确认**：对有歧义的润色，标注多个选项让作者选择
4. **逐层递进**：先润色对话，再优化动作和心理，最后整体打磨

## 禁用词清单

| 禁用词 | 原因 | 替代建议 |
|--------|------|---------|
| 突然 | 太突兀 | 删除或改用具体时间 |
| 瞬间 | 同上 | 改用具体动作 |
| 只见 | 老套路 | 直接描写场景 |
| 但见 | 同上 | 同上 |
| 不由得 | 心理描写滥用 | 用表情/动作代替 |
| 情不自禁 | 同上 | 同上 |
| 嘴角勾起 | 用烂的描写 | 具体神情 |
| 眸光一闪 | 同上 | 具体眼神变化 |
| 紧接着 | 节奏太赶 | 改用自然过渡 |

## 输出格式

```
【润色说明】
- 修改了哪些地方
- 为什么这样改
- 是否有歧义需要作者确认

【润色后内容】
正文内容...
```
