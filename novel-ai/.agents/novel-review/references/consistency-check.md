# 一致性检查提示词模板

用于 `novel-review` 技能的复核检查。

## 角色一致性检查

### 检查清单

```
【身份】
[] 角色名称是否前后一致
[] 角色身份是否与设定匹配

【性格】
[] 角色行为是否符合性格设定
[] 遇到冲突时的反应是否与性格一致
[] 成长变化是否有迹可循

【话风】
[] 对话是否符合角色话风
[] 是否有不同角色说话风格雷同的问题
[] 口头禅是否正确使用

【能力】
[] 能力展示是否与境界/等级一致
[] 是否有能力滥用/忽强忽弱的问题
[] 能力提升是否有合理过程

【关系】
[] 与其他角色的关系是否与设定一致
[] 关系变化是否有合理原因
[] 互动是否符合关系定位
```

## 设定一致性检查

### 检查清单

```
【力量体系】
[] 境界等级是否正确
[] 战斗结果是否符合实力逻辑
[] 修炼/升级过程是否合理

【世界观】
[] 世界规则是否前后一致
[] 地理/地点是否与设定匹配
[] 势力格局是否与设定一致

【物品/功法】
[] 物品属性是否前后一致
[] 功法描述是否矛盾
[] 物品归属是否有误
```

## 时间线检查

### 检查清单

```
【顺序】
[] 事件顺序是否合理
[] 因果关系是否成立

【跨度】
[] 时间跨度是否合理
[] 是否有时间跳跃矛盾

【季节/环境】
[] 季节描写是否矛盾
[] 环境变化是否合理
```

## 伏笔追踪检查

### 检查清单

```
【伏笔埋设】
[] 已埋伏笔是否记录在 tracking/plot-tracker.json
[] 伏笔是否自然埋入
[] 伏笔是否太明显或太隐蔽

【伏笔回收】
[] 已回收伏笔是否在 tracking 中标记
[] 回收方式是否合理
[] 是否有伏笔被遗忘

【伏笔一致性】
[] 伏笔细节前后是否一致
[] 伏笔与后续情节是否矛盾
```
