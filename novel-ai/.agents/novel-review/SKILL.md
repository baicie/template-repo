# 小说复核检查

## 触发场景

用户说"小说复核"、"章节检查"、"一致性检查"、"质量检查"、"小说质检"、"检查角色一致性"、"检查时间线"。

## 检查项

| 指令 | 检查内容 |
|------|---------|
| `--角色` | 只检查角色一致性 |
| `--时间线` | 只检查时间线 |
| `--设定` | 只检查设定一致性 |
| `--大纲` | 只检查大纲偏离 |
| `--伏笔` | 只检查伏笔回收 |


| `--文风` | 只检查文风一致性（含流水账检测） |
| `--开篇` | 只检查开篇质量（仅前3章） |
| `--全文` | 全面检查（所有项） |
| `--quick` | 快速检查（摘要模式） |
| `--report` | 生成完整报告 |

## 检查维度

### 1. 角色一致性

- 角色行为是否符合性格设定？
- 对话风格是否与角色一致？
- 能力边界是否前后统一？
- 关系变化是否有迹可循？

### 2. 时间线

- 事件顺序是否合理？
- 时间跨度是否一致？
- 是否有时间漏洞？

### 3. 设定一致性

- 力量体系是否统一？
- 世界观规则是否前后矛盾？
- 地理/势力设定是否一致？

### 4. 大纲偏离

- 当前内容是否偏离主线？
- 支线是否服务主线？
- 节奏是否符合大纲？

### 5. 伏笔回收

- 已埋伏笔是否回收？
- 回收方式是否合理？
- 是否有漏掉的伏笔？

### 6. 文风一致性

- 语言风格是否统一？
- 是否有流水账问题？
- 对话是否自然？

### 7. 开篇质量（前3章）

- 黄金三章标准
- 前300字是否有爆点
- 结尾是否有钩子
