# Consistency Check — 一致性检查

## 检查范围

### 1. 力量体系一致性
- 角色展现的实力是否与设定等级匹配？
- 能力的描述是否与体系规则矛盾？
- 战斗结果是否符合实力逻辑？
- 升级过程是否符合突破条件？

### 2. 角色行为一致性
- 角色的行为是否符合其性格设定？
- 对话风格是否与角色话风一致？
- 心理活动是否符合角色成长阶段？
- 在不同场景中行为是否一致？

### 3. 地理/时间一致性
- 场景描写是否与地理设定矛盾？
- 位置移动是否符合距离/时间逻辑？
- 时间线是否连贯？（不要出现"昨天"和"三天前"矛盾）

### 4. 逻辑漏洞
- 情节推进是否有明显逻辑问题？
- 是否有伏笔没有回收？
- 是否有"金手指"用完就忘的情况？
- 是否有前文信息后文矛盾？

### 5. 前后文矛盾
- 同一事件在不同章节的描述是否一致？
- 角色外貌/性格描写是否前后统一？
- 物品/能力的存在状态是否一致？

## 使用模板

```markdown
# 一致性检查

## 待检查章节
{{chapter_content}}

## 小说设定（从对应文件提取）
【世界观规则】{{world_rules}}
【力量体系】{{power_system}}
【角色设定】{{character_definitions}}
【地理设定】{{geography}}

## 检查维度
1. 力量体系一致性
2. 角色行为一致性
3. 地理/时间一致性
4. 逻辑漏洞
5. 前后文矛盾

## 输出格式
对每个发现的问题：
- 位置（如：第 3 段、第 5 行）
- 问题描述
- 严重程度：【严重/中等/轻微】
- 修改建议

最后给出总体评价：【优秀/良好/需修订】
```

## 输出示例

```
# 一致性检查报告

## 发现问题

### 问题 1
- **位置**：第 3 段，第 2 行
- **问题**：主角使用「裂空斩」，但「裂空斩」在第 15 章已设定为需要元婴境才能使用，而主角目前仍是金丹境
- **严重程度**：严重
- **修改建议**：将招式改为金丹境可用的「破风拳」，或在之前埋下主角已突破的伏笔

### 问题 2
- **位置**：第 7 段
- **问题**：主角说「我已经三年没见过师父了」，但第 42 章写的是「两年前在宗门见过一面」
- **严重程度**：中等
- **修改建议**：统一为「两年」，或修改前文时间线

### 问题 3
- **位置**：第 2 段
- **问题**：「他穿过繁华的街道」——但根据地理设定，主角所在的是偏远小镇，没有繁华街道
- **严重程度**：轻微
- **修改建议**：改为「穿过镇子中心的小巷」

## 伏笔追踪检查

| 伏笔 | 埋下章节 | 当前章节 | 状态 |
|------|---------|---------|------|
| 玄铁令来历 | 第 12 章 | 第 37 章 | 未回收 |
| 主角身世之谜 | 第 8 章 | 第 37 章 | 未回收 |

## 总体评价

【需修订】
该章节存在 1 个严重问题，需要修改后再进入下一章。2 个轻微问题可在后续润色时处理。
伏笔「玄铁令来历」已埋下 25 章，建议在第 40 章左右回收。
```

## 检查原则

1. **严格对照设定**：设定文件是唯一标准，不要凭感觉判断
2. **区分严重程度**：
   - **严重**：影响核心情节逻辑的问题
   - **中等**：影响阅读体验但可忽略的问题
   - **轻微**：纯文字/格式问题
3. **提出具体修改建议**：不要只说"有问题"，要给出解决方向
4. **更新伏笔追踪**：检查完成后，更新 `00-context.md` 中的伏笔追踪表

## 快速检查清单

对于简单的检查任务，使用快速清单：

```
[] 实力等级是否匹配
[] 招式名称是否正确
[] 势力名称是否一致
[] 角色对话是否贴合性格
[] 时间线是否连贯
[] 地理描写是否合理
[] 伏笔是否持续推进
[] 金手指使用是否合理
```

## 特殊场景检查

### 战斗场景
- 战斗结果是否符合实力对比？
- 招式消耗是否符合设定？
- 受伤程度是否合理？

### 情感场景
- 角色行为是否符合性格？
- 情感发展是否循序渐进？
- 对话风格是否统一？

### 信息交代
- 是否有重复交代已知的设定？
- 新信息是否与旧设定矛盾？
- 信息密度是否合适？
