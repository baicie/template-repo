# Inspiration — 灵感发散

## 使用场景

- 卡文，不知道下一步怎么写
- 需要一个冲突/反转/高潮
- 缺少一个有力的 boss 战设计
- 角色名字/地名/招式名不够满意
- 章节标题想不出来

## 1. 场景灵感 (scenario-gen)

获取特定场景的创意方向。

```markdown
# 场景灵感生成

## 当前状态（从 00-context.md 提取）
- 当前情节：{{current_plot}}
- 出场角色：{{active_characters}}
- 当前情绪/氛围：{{current_mood}}
- 章节类型：{{chapter_type}}

## 需要什么类型的场景
- 场景类型：{{scenario_type}}（战斗/情感/日常/转折/高潮）
- 需要解决的问题：{{problem_to_solve}}

## 生成方向
生成 3-5 个不同方向的场景灵感：
1. 方向 A：具体描述 + 核心冲突 + 可能的爽点
2. 方向 B：...
3. 方向 C：...

每个方向包含：
- 场景核心事件
- 关键冲突
- 预期效果
```

### 场景灵感方向库

**战斗场景方向**：
- 越级挑战（以弱胜强）
- 多人混战（乱中取胜）
- 保护某人（守护主题）
- 争夺资源（限时压力）
- 救援/逃亡（紧迫感）
- 势力对决（宏大场面）

**情感场景方向**：
- 误会与和解
- 表白/被告白
- 离别/重逢
- 误会加深
- 共同经历危机
- 日常互动升温

**转折场景方向**：
- 盟友背叛
- 意外发现真相
- 计划被打乱
- 获得关键信息/物品
- 身份暴露
- 绝境反转

## 2. 冲突设计 (conflict-gen)

设计核心冲突，让故事更有张力。

```markdown
# 冲突设计

## 已有冲突（从 00-context.md 提取）
- 主线冲突：{{main_conflict}}
- 当前阶段冲突：{{current_conflict}}

## 需要设计的冲突类型
- 冲突层级：{{level}}（个人/势力/世界）
- 冲突性质：{{nature}}（正邪/利益/理念/误会）

## 冲突要素
- 冲突双方：{{party_a}} vs {{party_b}}
- 核心矛盾：{{core_contradiction}}
- 冲突升级点：{{escalation_points}}

## 输出 3 个冲突方案
每个方案包含：冲突起点 → 发展 → 高潮 → 结果
```

### 冲突类型库

**个人层面**：
- 目标与能力的差距
- 感情与理智的矛盾
- 忠诚与正义的抉择
- 过去与现在的冲突

**势力层面**：
- 资源争夺
- 势力版图扩张
- 宗门/家族内斗
- 联盟破裂

**理念层面**：
- 正邪之争
- 改革与保守
- 个人与集体
- 复仇与放下

## 3. 命名灵感 (name-gen)

获取角色名/地名/招式名/势力名。

### 角色命名

```markdown
# 命名灵感

## 命名需求
- 类型：{{type}}（主角/反派/配角/势力名/地名/招式名）
- 性别：{{gender}}（仅角色）
- 风格：{{style}}（古风/现代/玄幻/仙侠/西方）
- 寓意：{{meaning}}（名字要体现什么）
- 已有名字：{{existing_names}}（避免重复）

## 生成要求
生成 10 个候选名，标注每个名字的特点和适用场景。
```

### 命名风格指南

**玄幻/仙侠类**：
- 单字名 + 双字名组合
- 带偏旁部首的字更有古感
- 避免太常见的名字
- 招式名要有画面感

**都市类**：
- 现代感，简洁有力
- 有记忆点
- 符合角色气质

## 4. 章节标题 (title-gen)

生成有吸引力的章节标题。

```markdown
# 章节标题生成

## 当前章节内容（从 00-context.md 提取）
- 章节核心事件：{{core_event}}
- 核心爽点：{{shuangdian}}
- 出场重要人物：{{important_chars}}
- 章节类型：{{chapter_type}}

## 生成要求
生成 5 个章节标题候选：
- 1-2 个简洁有力型
- 1-2 个有悬念感型
- 1-2 个有爽感型

每个标题包含：标题 + 副标题（可选）+ 理由
```

### 标题风格库

**简洁有力型**：
- 「一剑破空」
- 「元婴突破」
- 「宗门大比」
- 「绝地反击」

**悬念感型**：
- 「他到底是谁」
- 「隐藏在暗处的敌人」
- 「三年之约」
- 「真相大白」

**爽感型**：
- 「碾压局」
- 「打脸时刻」
- 「力挽狂澜」
- 「一战成名」

## 灵感使用流程

```
1. 明确当前困境
   ↓
2. 读取 00-context.md 了解当前状态
   ↓
3. 选择需要的灵感类型
   ↓
4. 生成多个方向
   ↓
5. 选择最喜欢的方向
   ↓
6. 深挖该方向，补充细节
   ↓
7. 确定后更新 00-context.md
```

## 原则

1. **先方向后细节**：先生成多个方向，选定后再深入
2. **结合已有设定**：灵感要与已有设定兼容，不能天马行空
3. **服务于情节**：灵感是为了推动情节，不是为了新奇
4. **保留选择权**：给出多个选项，让作者做最终决定
