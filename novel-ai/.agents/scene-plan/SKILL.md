# 场景规划

## 触发场景

用户说"场景规划"、"规划场景"、"场景设计"、"场景清单"、"规划章节"、"批量生成"。

## 场景规划流程

### 步骤1：生成场景清单

基于大纲，将每章拆解为1-3个核心场景。

### 步骤2：场景分类

| 类型 | 说明 | 特征 |
|------|------|------|
| 主动场景 | 主角有目标，主动行动 | 有目标→行动→结果 |
| 被动场景 | 主角被迫应对 | 意外→应对→结果 |
| 过渡场景 | 连接两个主动场景 | 简短，不要太长 |

### 步骤3：场景规划详情

每个场景包含：
- 场景类型
- 核心目标
- 涉及角色
- 关键冲突
- 场景结果
- 情绪走向

## 场景规划模板

### 主动场景模板

```
## 场景：{{场景名}}

- **类型**：主动场景
- **章节**：第X章
- **核心目标**：主角想要达成什么？
- **涉及角色**：{{角色列表}}
- **关键冲突**：遇到什么阻碍？
- **场景结果**：达成/部分达成/失败
- **情绪走向**：期待→紧张→满足/失望
- **字数**：约{{字数}}字
```

### 被动场景模板

```
## 场景：{{场景名}}

- **类型**：被动场景
- **章节**：第X章
- **核心事件**：发生了什么意外？
- **涉及角色**：{{角色列表}}
- **主角应对**：主角如何应对？
- **场景结果**：化险为夷/付出代价/有悬念
- **情绪走向**：震惊→紧张→转机
- **字数**：约{{字数}}字
```

## 批量场景规划

支持指定章节范围：
- `/scene-plan` — 全量规划
- `/scene-plan 第3章` — 局部规划
- `/scene-plan 第3-5章` — 指定范围
