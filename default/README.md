# Default 项目模板

这是一个简洁的 Node.js + TypeScript 项目模板，可作为新项目的起点。

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建项目
pnpm build

# 运行构建后的代码
pnpm start

# 监听模式（自动重启）
pnpm watch

# 清理构建产物
pnpm clean
```

## 项目结构

```
default/
├── src/
│   └── index.ts      # 入口文件
├── package.json      # 项目配置
├── tsconfig.json     # TypeScript 配置
└── README.md         # 项目说明
```
